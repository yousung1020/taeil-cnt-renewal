# 프로젝트 설명
- 해당 프로젝트는 '태일씨앤티' 회사의 웹사이트를 리뉴얼하기 위한 프로젝트입니다.
- 기존 홈페이지 링크는 "http://www.taeilcnt.co.kr/home/index.jsp" 입니다.
- 태일씨앤티는 철근, 콘크리트 전문 건설 회사입니다.
- 이 사이트의 주 사용자는 홈페이지를 통해 태일씨앤티의 지원하려는 사람이며, 개발한 사이트를 심사하는건 50~60대 연령층의 회사 직원 분들입니다. 이를 고려하여 UI/UX를 설계해야 합니다.

# 개발 관련 설명
- 본 프로젝트는 React + TypeScript + Tailwind CSS를 사용하여 개발합니다.
- 이 사이트는 기존 회사 정책에 따라 정적으로 호스팅할 예정이며, 동적인 기능이 포함되어서는 안됩니다. 서버리스 환경에서 동작해야 합니다.
- 이미지 및 기타 에셋들은 절대경로가 아닌 상대경로로 작성합니다.
- 태일씨앤티 로고 및 홈페이지의 주요 컬러는 초록색입니다. 정확한 컬러코드는 #55B76F(연한 초록), #40776D(진한 초록)이며 UI를 만들때 해당 색상을 참고합니다.
- 해당 사이트에 사용되는 폰트나 이미지는 저작권 문제가 없어야 하며, 상업적으로 이용이 가능해야 합니다.
- 개발할 때 하나의 tsx 파일에 모든 코드를 다 작성하지 않고 컴포넌트를 나누어서 작성합니다.
- 영상과 이미지는 WebM/WebP 등 최적화해서 로딩 속도를 잡아주세요.
- 모바일, 태블릿, PC 어디서든 화면이 깨지지 않는 반응형 웹으로 구현해야합니다.
- 디자인 컨셉: “Trust & Professional”(신뢰감, 전문적인)
